# 康养医卫短考项目知识库策划模板

## 覆盖项目

适用于健康管理师、公共营养师、心理咨询师、家庭教育指导、养老照护、老年人能力评估、长期照护、中医康复、中医膳食、食品安全管理、执业药师及其他康养医卫短考或技能认证项目。

成熟康养知识库通常会同时包含：项目体系、证书口径、报考条件、考试政策、课程大纲、服务体系、政策市场、应用场景、补贴或职业场景、推荐指南、销售答复卡、异议处理、沟通脚本、跨项目推荐、客户可读 FAQ 和月度风险口径。

## 策划定位

康养医卫项目的策划重点不是把证书包装成“万能收益工具”，而是把它放在健康服务、岗位能力、家庭自用、职业转型和政策场景中，讲清楚谁适合学、学了能做什么、边界在哪里。

## 产品背景素材

| 层级 | 可用表达 |
|---|---|
| 市场层 | 健康管理、营养指导、心理服务、养老照护、慢病管理和社区服务需求持续增长，相关岗位更重视复合型服务能力。 |
| 用户层 | 客户常见诉求包括职业转型、岗位加分、家庭健康管理、副业探索、证书储备和专业能力提升，但也容易被补贴、兼职和收益承诺误导。 |
| 产品层 | 产品应通过基础理论、案例实操、服务场景训练、题库练习、证书报考指导和学习陪跑，帮助学员建立可落地的健康服务能力。 |

## 适合人群拆分

| 人群 | 动机 | 产品承接 |
|---|---|---|
| 医护、药店、体检、养老、母婴等在岗人员 | 岗位能力和服务增值 | 专业提升班、案例训练、证书报考指导 |
| 想转入健康服务行业的人 | 转行和职业探索 | 入门系统班、行业场景课、就业/岗位认知边界 |
| 宝妈、家庭照护者、健康爱好者 | 自用和家庭管理 | 低门槛基础班、生活场景案例、客户可读素材 |
| 想做副业或私域服务的人 | 增收探索 | 服务场景训练、内容表达、合规收益边界 |
| 已学营养/心理/健康单项目的人 | 复合能力 | 关联项目组合推荐，如营养+健康、心理+家庭教育 |

## 产品班型字段

- 班型：基础入门班、精品系统班、实操提升班、协议/保障班、组合项目班。
- 课程：基础理论、行业政策、案例分析、实操训练、题库模考、服务场景演练。
- 服务：班主任、答疑、题库、学习计划、资料讲义、考试提醒、证书查询说明。
- 证书：发证单位、证书性质、查询方式、适用边界、地区政策差异。
- 应用：岗位能力提升、家庭健康管理、社群服务、养老/母婴/药店/体检机构场景。
- 风险：补贴、兼职收入、证书含金量、就业承诺、项目授权和报考资格需确认。

## 主卖点方向

| 方向 | 适用项目 | Slogan 思路 |
|---|---|---|
| 场景实用 | 营养、健康、心理、养老 | 学得会，也用得上 |
| 职业复合 | 医护药店、体检、养老从业者 | 多一项能力，多一类服务 |
| 家庭自用 | 营养、心理、家庭教育 | 先照顾好家人，再提升自己 |
| 系统陪跑 | 短考证书类 | 从报考到备考，有人带着走 |
| 必学必要性 | 政策或岗位驱动项目 | 不是包结果，是把能力补齐 |

## 产品优势写法

### 课程体系

建议写成“基础认知 -> 专业知识 -> 案例实操 -> 题库训练 -> 场景应用”。短考项目不能只有刷题，还要解释课程如何解决真实服务场景。

### 服务体系

可突出：

- 报考条件和材料提醒。
- 阶段学习计划。
- 案例答疑和实操点评。
- 考试节点提醒。
- 证书查询和使用边界说明。

### 政策和市场

可以写“行业需求增长、岗位服务升级、社区健康服务发展”等趋势，但不写“国家强制所有人必考”或“拿证必有工作”。

## 异议转策划素材

| 异议 | 策划补强 |
|---|---|
| 这个证有没有用 | 增加证书性质、应用场景、适合人群和不可承诺边界 |
| 能不能兼职赚钱 | 增加技能应用和服务场景，但标注收益取决于个人资源和当地市场 |
| 能不能领补贴 | 增加补贴查询和地区差异，不能承诺必领 |
| 我不是相关专业 | 增加入门路径、学习门槛和报考资格待确认 |
| 自己看书行不行 | 增加案例训练、答疑和学习节奏价值 |

## 产介话术模板

一句话介绍：

> 这个项目适合想把健康、营养、心理或照护知识系统学一遍的人，不只是为了考证，也能帮你理解真实服务场景和证书使用边界。

私聊承接：

> 我先确认一下，您学这个是为了岗位需要、转行尝试，还是家庭自用？三个方向推荐的课程深度不一样，证书和实操内容也要分开看。

## 合规边界

- 不承诺证书等同职业资格、执业资格或全国统一岗位准入。
- 不承诺兼职收益、就业、高薪、补贴必领。
- 不夸大发证单位和证书效力。
- 不把“必学/必考”说成所有客户都必须报名，应解释为适配人群值得系统规划。
