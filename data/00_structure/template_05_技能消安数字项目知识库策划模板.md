# 技能消安数字项目知识库策划模板

## 覆盖项目

适用于 PLC 智能制造、自动化工程师、软考、无人机、物业管理、应急救援、消防设施操作员/消防安全相关、AIGC 数字技能、办公技能和其他实操训练类项目。

成熟技能类知识库除了产品、课程和证书，还会沉淀：实操训练路径、就业/岗位案例边界、设备或软件环境、政策市场、服务体系、FAQ 知识卡、行业转化打法、场景异议和“必学/系统训练必要性”合规话术。

## 策划定位

技能类产品策划不能只写“学完会某技能”。要讲清楚：客户从什么基础出发，要达到什么工作或应用场景，课程如何从认知、操作、项目练习到案例输出逐步训练。就业、接单、涨薪等只能作为可能场景，不能承诺结果。

## 产品背景素材

| 层级 | 可用表达 |
|---|---|
| 市场层 | 智能制造、数字化办公、低空经济、物业服务、应急管理和信息化岗位对实操能力、证书资质或项目经验提出更高要求。 |
| 用户层 | 客户常见问题是零基础怕学不会、转行怕没项目经验、在岗想提效、想考证但不知道证书和能力如何衔接。 |
| 产品层 | 产品应通过基础理论、工具操作、实操项目、案例训练、答疑辅导、作品/案例输出和岗位场景说明，帮助客户把技能学到可展示、可复盘。 |

## 适合人群拆分

| 人群 | 动机 | 产品承接 |
|---|---|---|
| 零基础转行者 | 想进入新岗位 | 入门班、项目实训、岗位认知、风险边界 |
| 在岗提升者 | 提效、晋升、拓展能力 | 进阶班、真实案例、专项模块 |
| 证书刚需者 | 单位要求或岗位门槛 | 报考指导、考试训练、证书查询边界 |
| 副业探索者 | 接单或商业化尝试 | 案例训练、作品输出、收益不承诺 |
| 跨行业客户 | 原行业转数字/技能方向 | 行业转化路径、基础补课、作品/项目沉淀 |

## 产品班型字段

- 班型：基础班、进阶班、实操班、证书班、就业/项目实训班、线上轻量班。
- 课程：基础理论、工具软件、案例拆解、项目实操、考试训练、作品输出。
- 服务：答疑、设备/软件指导、班主任、项目点评、就业/岗位认知、资料和练习。
- 证书或资质：报考条件、考试科目、证书查询、适用边界。
- 案例：成绩单、好评、作品、项目案例只能做信任素材，不能承诺就业和收入。
- 风险：设备条件、地区政策、证书效力、就业推荐、兼职收益必须确认。

## 主卖点方向

| 方向 | 适用项目 | Slogan 思路 |
|---|---|---|
| 实操项目 | PLC、无人机、AIGC、物业 | 不只听课，跟着项目练出来 |
| 零基础友好 | 数字技能、软考、消防 | 从入门到能做，节奏拆清楚 |
| 岗位转化 | PLC、软考、物业、应急 | 围绕岗位场景学技能 |
| 证书+能力 | 消防、软考、应急 | 证书是入口，能力才是底气 |
| 系统训练必要性 | 高实操项目 | 别靠碎片课碰运气，按流程练 |

## 产品优势写法

### 课程体系

建议写成“认知导入 -> 工具/规范 -> 分步操作 -> 项目案例 -> 复盘点评 -> 场景应用”。技能类项目尤其要写清实操任务和作品输出。

### 服务体系

可突出：

- 工具安装和学习环境指导。
- 项目作业点评。
- 真实案例拆解。
- 考证节点提醒。
- 岗位能力说明和简历/作品边界。

### 行业转化

软考可写 IT、系统集成、建工、消防、医卫、教育等跨行业转化；PLC 可写制造业、自动化、电工和设备维护；无人机可写巡检、航拍、测绘等场景。但具体就业和收入必须待确认。

## 异议转策划素材

| 异议 | 策划补强 |
|---|---|
| 零基础怕学不会 | 增加入门课、工具演示、分步项目和答疑 |
| 学了能不能就业 | 增加岗位认知、作品案例和就业边界 |
| 有没有实操 | 增加设备/软件环境、实训任务和项目成果 |
| 只想考证 | 增加证书政策、考试训练和能力应用 |
| 价格贵 | 增加服务清单、实操资源和长期能力价值 |

## 产介话术模板

一句话介绍：

> 这个项目不是单纯看视频，而是按岗位场景把工具、案例和实操任务拆开练，适合想从零基础补技能或在岗提升的人。

私聊承接：

> 您现在是想考证、转行，还是在现有岗位上提升技能？如果是转行，我们先看基础和可投入时间；如果是岗位提升，就重点看实操模块和项目案例。

## 合规边界

- 不承诺就业、高薪、兼职接单收益、入专家库或退休金增加。
- 不承诺证书一定带来补贴、落户、职称或招投标结果。
- 不夸大设备、软件、授权和实操条件。
- 不把“必学”讲成所有人必须报名，应解释为匹配岗位和学习目标的人值得系统训练。
