# 考研公考招录项目知识库策划模板

## 覆盖项目

适用于国考、省考、事业单位、教师资格证、法考、考研、同等学力申硕、党校在职研等招录、升学和资格考试项目。

成熟考研公考知识库强调“全资料知识库化”：产品手册、备考规划、岗位表、院校专业表、真题题本、试听链接、群发话术、选岗流程、报名流程和内部资料都要转成范围、用途、判断边界、销售承接和客户可读表达。

## 年度映射

这类项目必须区分资料年份、产品年份和目标考试届别。

示例：2026 年整理的国考资料可能服务 2027 国考备考；2026 年考研产品可能面向 2027 考研。策划方案里不要仅凭“2026”判断资料过期，应写“当前产品年份/资料年份/目标考试届别以项目部口径为准”。

## 策划定位

考研公考项目的核心不是简单卖课，而是帮助客户完成“目标判断 -> 路径规划 -> 资料/课程/训练 -> 节点管理 -> 风险兜底”。产品策划要突出规划感、陪跑感和阶段训练，而不是只写课时和价格。

## 产品背景素材

| 层级 | 可用表达 |
|---|---|
| 市场层 | 招录竞争、岗位筛选、学历提升、职业稳定性和选拔考试难度，让系统规划比临时刷题更重要。 |
| 用户层 | 客户常见问题是目标模糊、基础薄弱、时间不足、岗位/院校选择困难、资料多但不会用、担心错过报名和考试节点。 |
| 产品层 | 产品应整合课程、题库、模考、批改、选岗/择校指导、备考规划、节点提醒和班主任服务，形成完整备考闭环。 |

## 适合人群拆分

| 人群 | 痛点 | 产品承接 |
|---|---|---|
| 应届生 | 时间相对集中但目标和路径不清 | 全程规划、基础系统课、择校/选岗指导 |
| 在职备考人 | 时间碎片化，容易断学 | 阶段计划、短课节、督学提醒、周测模考 |
| 二战三战考生 | 有经验但短板明显 | 诊断测评、专项突破、模考复盘、批改服务 |
| 跨专业/跨岗人群 | 不知道是否适合 | 入门导学、资格边界、目标风险评估 |
| 只想要资料的人 | 资料多但不会用 | 资料使用方法、训练路径、课程承接 |

## 资料类素材如何转策划

| 资料 | 产品策划使用方式 |
|---|---|
| 岗位表 | 写成选岗服务、资格判断、信息采集和风险边界 |
| 院校专业表 | 写成择校择专业服务、目标匹配和年度风险 |
| 真题题本 | 写成题型认知、训练路径、模考复盘和阶段提分 |
| 备考规划 PDF | 写成学习阶段、分科策略、每日/每周安排和顾问诊断 |
| 试听链接和群发话术 | 写成试听课匹配逻辑、私域转化节奏和禁用表达 |
| 报名流程资料 | 写成节点提醒、材料清单、常见错误和官方优先边界 |

## 产品班型字段

- 班型：基础系统班、全程协议班、专项突破班、冲刺模考班、面试班、择校/选岗服务包。
- 课程：基础、强化、刷题、模考、冲刺、面试、论文/申论/专业课批改。
- 服务：班主任、答疑、作文/申论/论文批改、选岗/择校咨询、资料规划、督学、节点提醒。
- 资料：真题、题本、讲义、岗位表、院校表、规划表；只能写使用方法，不承诺发原始附件。
- 风险：岗位资格、院校政策、报名条件、分数线、招录人数、上岸结果都不能承诺。

## 主卖点方向

| 方向 | 适用项目 | Slogan 思路 |
|---|---|---|
| 全程规划 | 考研、公考、事业单位 | 从目标到上考场，一路有人带 |
| 选岗择校 | 国考、省考、考研 | 先选对目标，再开始冲刺 |
| 分科突破 | 行测申论、考研专业课 | 短板不硬扛，按科目拆解练 |
| 在职陪跑 | 在职考研、公考 | 工作再忙，也有清晰节奏 |
| 资料使用 | 只要资料客户 | 资料不是堆起来，是用起来 |

## 产介话术模板

一句话介绍：

> 这个班不是只给一堆课，而是把目标选择、阶段学习、题目训练、模考批改和报名节点放在一个节奏里，适合想系统备考的人。

私聊承接：

> 您现在最需要先判断目标：是岗位/院校还没定，还是已经定了但不知道怎么学？如果目标没定，先做选岗或择校诊断；如果目标已定，再看全程班还是专项强化。

## 合规边界

- 不承诺上岸、录取、过线、内部名额或押题命中。
- 不把历史岗位、分数线、院校专业变化说成最新。
- 不提供原始附件发送承诺，只输出整理后的资料范围、用途和风险边界。
- 年份、地区、岗位、院校、报名时间和政策必须以官方公告与项目部最新口径为准。
