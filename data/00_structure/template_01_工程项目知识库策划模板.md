# 工程项目知识库策划模板

## 覆盖项目

适用于一级建造师、二级建造师、一级造价工程师、二级造价工程师、监理工程师、咨询工程师、中级安全工程师、一级消防工程师、中级经济师、水利五大员、九大员等建筑工程长考类项目。

参考成熟知识库特征：工程项目通常同时沉淀 `overview`、`products`、`course-syllabus`、`exam-policy`、`exam-analysis`、`certificate`、`service-system`、`application-scenarios`、`policy-and-market`、`teachers-and-brand-proof`，销售层有推荐指南、答复卡、异议处理、沟通脚本、风险控制、跨项目联动和“必考必过”合规转译。

## 策划定位

工程类产品策划不要只写“考证培训”。它的核心是：在职工程人时间碎片化、考试周期长、科目多、政策和证书价值复杂，需要一个能把备考路径、专业选择、课程服务和风险边界讲清楚的系统方案。

## 产品背景素材

| 层级 | 可用表达 |
|---|---|
| 市场层 | 工程行业资质监管、项目招投标、企业持证需求、四库一平台、职称和执业资格联动，使证书仍是职业发展中的重要筹码。 |
| 用户层 | 在职工程人普遍工作强度高、学习时间碎片化、法规和管理科目容易拖后腿，实务方向选择和备考节奏也常常缺少判断。 |
| 产品层 | 产品应通过阶段化课程、重点考点梳理、实务专项、题库模考、督学答疑和协议服务，帮助学员把长周期备考拆成可执行任务。 |

## 适合人群拆分

| 人群 | 核心痛点 | 产品承接 |
|---|---|---|
| 首考在职工程人 | 不知道从哪学，担心一次投入太大 | 基础系统班、学习规划、科目节奏表 |
| 二战三战考生 | 差几分、实务不过、方法疲劳 | 强化冲刺、实务专项、错题诊断、模考复盘 |
| 已有低阶证书者 | 想升级一建、一造、监理或中安 | 证书进阶路径、专业选择、跨项目联动 |
| 企业资质相关人员 | 公司需要持证、投标或岗位要求 | 报考条件核验、学习进度管理、证书应用边界 |
| 想转工程管理岗的人 | 缺少行业经验或证书背书 | 入门导学、政策市场说明、职业路径风险提示 |

## 产品班型设计字段

工程类产品至少要补齐：

- 班型：基础班、精品班、协议班、高端班、冲刺班、单科班。
- 科目：法规、管理、经济、案例/实务；不同项目按真实科目替换。
- 实务方向：建筑、市政、机电、公路、水利、矿业等，需提示以项目实际开设为准。
- 服务：班主任、答疑、题库、模考、作业批改、直播回放、资料包、考前密训。
- 保障：协议、续学、退费、重学等必须写“按当期协议执行，待确认”。
- 专业选择：根据从业方向、单位资质、备考基础和地区需求判断，不能直接替客户承诺收益。

## 主卖点方向

| 方向 | 适用项目 | Slogan 思路 |
|---|---|---|
| 在职备考陪跑 | 一建、二建、造价、监理 | 工程人备考，有人规划有人管 |
| 实务突破 | 一建、一消、中安、监理 | 实务不过关，就按场景拆着练 |
| 协议保障 | 长考协议班 | 学得完、有人问、结果更稳 |
| 专业选择 | 一建、中安、中经 | 先选对方向，再开始备考 |
| 企业刚需 | 建工证书类 | 围绕岗位和资质，把证书用起来 |

## 产品优势写法

### 教学体系

建议按“基础精讲 -> 强化刷题 -> 案例/实务专项 -> 模考复盘 -> 考前冲刺”写。不要只写课时多，要说明每个阶段解决什么问题。

### 服务保障

工程学员常见困难是忙、断更、无人答疑。服务优势可写：

- 每周学习计划和进度提醒。
- 重点科目答疑和错题复盘。
- 考前节点提醒。
- 实务方向/专业选择建议。

### 品牌和师资

可从教研年限、工程项目经验、考试命题规律研究、教材更新、案例讲解能力写，但具体老师姓名、职称、通过率必须待确认。

## 销售异议转策划素材

| 异议 | 策划反向补强 |
|---|---|
| 没时间学 | 强调短课节、阶段计划、碎片化学习、班主任督学 |
| 太贵了 | 强调服务清单、协议边界、长期职业价值，避免直接算高收益 |
| 自学也可以 | 强调考试范围广、实务主观题、错题反馈和学习路径 |
| 怕不过 | 强调模考、答疑、督学、复盘和协议/续学边界 |
| 不知道考哪个专业 | 增加专业选择测评和顾问诊断问题 |

## 产介话术模板

一句话介绍：

> 这个班主要是给在职工程人设计的，把长周期备考拆成阶段课程、题库训练、实务专项和督学服务，适合工作忙但想系统拿证的人。

私聊承接：

> 您先不用急着选班型，我先确认三个信息：今年是首考还是二战，打算报哪个专业方向，平时每天能稳定学多久。确认后再看是系统班、协议班还是单科强化更适合。

## 合规边界

- 不能承诺拿证后一定升职、涨薪、挂靠、兼职收益或单位补贴。
- 不能承诺包过、押题命中、内部题。
- 补贴、职称、资质、注册和地区政策必须以官方与项目部最新口径为准。
- 协议、退费、重学、续学以当期合同为准。
