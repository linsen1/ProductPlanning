# 合规边界库 · 全局红线

> 用途：每次生成方案时全局注入。所有高风险表达必须转为可替代表达。本库优先级最高，与任何其他层冲突时以本库为准。

## 一、禁止直接承诺（红线）

- 包过、保过、必过、不过一定退费。
- 押题命中、内部题、原题。
- 就业高薪、兼职必有收入、证书必然变现。
- 补贴必领、落户必成、职称必评。
- 官方授权 / 证书效力 / 报名资格 未经确认即断言。
- 把"必学/必考"说成所有客户都必须报名。
- 证书等同执业资格、职业资格或全国统一岗位准入。
- 夸大发证单位与证书效力（如混说"卫健委发证""国家执业资格"）。

## 二、可替代表达（对照）

| 想表达 | 合规说法 |
|---|---|
| 包过/押题 | 通过系统课程、督学与模考，提高备考确定性 |
| 拿证就能就业 | 证书是能力背书与起点，就业取决于个人与市场 |
| 兼职必赚 | 技能可用于多种服务场景，收益取决于个人资源与当地市场 |
| 补贴必领 | 部分地区有技能提升补贴，需查询当地目录，存在地区差异 |
| 国家强制必考 | 适合有明确目标、岗位或自用需求、能配合学习计划的人群 |
| 证书全国通用执业 | 职业技能等级/培训评价类证书，体现岗位能力，非执业准入 |

## 三、统一兜底句

- "退费、续学、补贴、报考资格以当期协议、官方公告和项目部口径为准。"
- 信息不足处统一标注："（参考，待策划人确认）"。

## 四、行业专项提示

- 营养/健康管理：膳食与健康干预属健康指导，不替代临床诊疗，不承诺疗效。
- 心理：心理服务非医疗诊疗，严重问题应转介专业医疗机构，不承诺独立执业。
